IMPORTANT INFO FOR TILESET USE
------------------------------
Tileset needs to have the following changes from default (inspector):
  - Sprite Mode 		: Multiple
  - Pixels per Unit 	: 200
  - Filter Mode		: Point (No filter)
  - Compression		: None

In sprite editor, slice all tilesets with the following:
  - Type			: Grid by Cell size
  - Pixel Size X		: 16
  - Pixel Size Y		: 16

All tiles created from the tilesets have been applied to a tile palette:
  - Palette Name		: TopDown Map Palette
  - Tile Folders		: Grass
			: Plants
			: Player
			: Props
			: Stone
			: Struct
			: Wall
